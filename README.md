# 🍽️ Zomato Restaurant Analysis

End-to-end data analysis project exploring restaurant performance on Zomato,
focused on online ordering, table booking, pricing, and market positioning.

- Features: rating, votes, cost for two, online order, table booking, restaurant type

## 🔍 Key Insights
- Online ordering and table booking are both statistically linked to higher ratings (p<0.05)
- Price has only a weak correlation with rating (r=0.275)
