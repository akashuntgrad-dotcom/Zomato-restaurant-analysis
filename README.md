# 🍽️ Zomato Restaurant Analysis

End-to-end data analysis project exploring restaurant performance on Zomato,
focused on online ordering, table booking, pricing, and market positioning.

## 📌 Business Problem
[1-2 sentences — e.g. "Does enabling online ordering/table booking 
correlate with better restaurant performance on Zomato?"]

## 📊 Dataset
- 148 restaurants from Zomato
- Features: rating, votes, cost for two, online order, table booking, restaurant type

## 🔍 Key Insights
- Online ordering and table booking are both statistically linked to higher ratings (p<0.05)
- Price has only a weak correlation with rating (r=0.275)
- [3-5 of your strongest insights as bullet points]

## 🛠️ Tools Used
Python, Pandas, Matplotlib, Seaborn, SciPy, Scikit-learn, Jupyter Notebook

## 📁 Project Structure
- `Zomato_Analysis.ipynb` — focused analysis on digital feature adoption
- `Zomato_Multi_Lens_Analysis.ipynb` — multi-angle analysis (format, pricing, clustering, opportunity scoring)

## 🚀 How to Run
1. Clone the repo
2. `pip install -r requirements.txt`
3. Open the notebook in Jupyter and run all cells
